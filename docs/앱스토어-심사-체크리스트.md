## 1. Apple Developer / Xcode (서명·기능)

- [ ] Xcode > Signing & Capabilities: 팀 선택, 번들 ID `com.singlemarks.timelock` 확인
- [ ] **Sign in with Apple** capability 추가 (Google 로그인 제공 시 심사 필수 — 4.8)
- [ ] **Time Sensitive Notifications** capability 추가 (알람이 집중 모드를 일부 관통)
- [ ] 버전 1.0.0 / 빌드 번호 확인 후 Archive → App Store Connect 업로드

- [ ] 부제(선택): "알람을 끄는 유일한 방법, 촬영 시작"
- [ ] 카테고리: 생산성 (보조: 라이프스타일)
- [ ] 연령 등급 설문: 전부 "없음" → 4+ 예상
- [ ] **개인정보처리방침 URL**: `https://singlemark.notion.site/39f41b10f64b80d2acaffcb5815106a9`

- [ ] 스크린샷 (6.7"/6.1" 필수): 홈 / 알람 화면 / 세션 시계판 / 기록 캘린더 / 페이월 — 5장 권장

## 3. App Store Connect — 앱 개인정보 (영양성분표)

- [ ] "데이터 수집" 항목 입력:
  - **연락처 정보 > 이메일 주소** — 수집됨 / 계정 기능에 사용 / 사용자 신원에 연결됨 / 추적 없음

  - 그 외(위치·사진·검색기록 등) 전부 "수집 안 함"

## 5. Firebase 콘솔

- [ ] Authentication > 템플릿: 보낸사람 이름 "앵그리모티 (AngryMoti)"로 변경
- [ ] Firestore > 규칙: 본인 문서만 읽기/쓰기 배포
  ```
  rules_version = '2';
  service cloud.firestore {
    match /databases/{database}/documents {
      match /users/{uid}/{document=**} {
        allow read, write: if request.auth != null && request.auth.uid == uid;
      }
    }
  }
  ```
- [ ] (권장) API 키에 iOS 번들 ID 제한 + App Check 활성화

> 이 앱은 사용자가 스스로 예약한 활동 시각에 알람을 울리고, 알람 해제를 위해 사용자가 직접 전면 카메라 타임랩스 촬영을 시작하는 자기관리 앱입니다. 모든 영상은 사용자 기기에만 저장되며 서버로 전송되지 않고, 저장하지 않은 영상은 자동 삭제됩니다. 테스트 방법: 로그인 후 홈에서 "지금 바로 시작" → 활동명 입력 → 10분 선택 → 촬영 준비하기 → 체크 후 촬영 시작. (짧은 테스트를 위해 10분 세션을 권장합니다)
